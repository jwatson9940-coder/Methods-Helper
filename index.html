The Power Rule

Integration reverses differentiation. To integrate a power of $x$, raise the index by 1 and divide by the new index.

$$\int x^n \, dx = \frac{x^{n+1}}{n+1} + c \quad (n \neq -1)$$

Always rewrite first: $\frac{1}{x^3} = x^{-3}$, and $\sqrt{x} = x^{1/2}$, before applying the rule.

∬ Medium

Standard Trig & Exponential Integrals

These results must be automatic — they appear inside almost every harder integration question.

$$\int e^x \, dx = e^x + c \qquad \int \sin x \, dx = -\cos x + c \qquad \int \cos x \, dx = \sin x + c$$

Common mix-up: the integral of $\sin x$ has a minus sign, but the integral of $\cos x$ does not. This is the opposite of what many people first guess.

∫ Easy
∬ Medium
∭ Hard
Reverse Chain Rule — Integrating Composite Functions

When the inner function of a composite is linear ($ax+b$), integrate as normal then divide by $a$. This is the most commonly tested integration skill in WACE papers.

$$\int f(ax+b)\,dx = \frac{1}{a}F(ax+b) + c \quad \text{where } F'(x) = f(x)$$

This only works directly when the inner function is linear. For non-linear inner functions (like $e^{x^2}$), you generally cannot integrate by inspection — that needs different techniques beyond this course.

∫ Easy
∬ Medium
∭ Hard
The Fundamental Theorem & Signed Area

A definite integral evaluates to a number — the signed area between the curve and the x-axis.

$$\int_a^b f(x)\,dx = F(b) - F(a) \quad \text{where } F'(x) = f(x)$$

Signed vs unsigned: if a curve dips below the axis, that region subtracts from the integral. For total (unsigned) area, split at the x-intercepts and add the absolute value of each piece.

∫ Easy
∬ Medium
∭ Hard
Area Between Curves & Real Applications

The hardest WACE integration questions combine area/kinematics with a "show that" or multi-part structure.

$$\text{Area} = \int_a^b \big[f(x) - g(x)\big]\,dx \quad \text{where } f(x) \geq g(x) \text{ on } [a,b]$$
AREA BETWEEN CURVES
Find intersections first. Then integrate (top − bottom).
KINEMATICS
$a(t)\xrightarrow{+c_1}v(t)\xrightarrow{+c_2}x(t)$ — two integrations, two conditions.

The Power Rule

To differentiate a power of $x$, multiply by the index, then reduce the index by 1.

$$\frac{d}{dx}(x^n) = nx^{n-1}$$

Remember: the derivative of a constant is always $0$, and the derivative of $kx$ (linear term) is always just $k$.

Standard Exponential & Trig Derivatives

These results are used constantly inside bigger product/chain rule problems — they must be automatic.

$$\frac{d}{dx}(e^x) = e^x \qquad \frac{d}{dx}(\sin x) = \cos x \qquad \frac{d}{dx}(\cos x) = -\sin x$$

Common mix-up: it's $\cos x$ that picks up the negative sign when differentiated, not $\sin x$. Many students remember it backwards.

d/dx Medium

Product Rule & Quotient Rule

Use the product rule when two functions are multiplied; use the quotient rule when one function is divided by another.

$$\frac{d}{dx}(uv) = u'v + uv'$$
$$\frac{d}{dx}\left(\frac{u}{v}\right) = \frac{u'v - uv'}{v^2}$$

Tip: always clearly label $u$, $v$, $u'$, $v'$ before substituting into the formula — this avoids sign and order mistakes, especially in the quotient rule.

d/dx Easy
d/dx Medium
d/dx Hard
The Chain Rule

Use the chain rule whenever you're differentiating a function inside another function — a composite function.

$$\frac{d}{dx}\big[f(g(x))\big] = f'(g(x)) \cdot g'(x)$$

Process: differentiate the outer function (leaving the inner untouched), then multiply by the derivative of the inner function.

d/dx Easy
d/dx Medium
d/dx Hard
Second Derivative, Stationary Points & Optimisation

The second derivative tells you about concavity, and stationary points are found by setting the first derivative to zero.

$$f'(x) = 0 \Rightarrow \text{stationary point} \qquad f''(x) > 0 \Rightarrow \text{local min} \qquad f''(x) < 0 \Rightarrow \text{local max}$$
SECOND DERIVATIVE TEST
Find $f'(x)=0$, then check the sign of $f''(x)$ at that point.
OPTIMISATION
Build one function of one variable, differentiate, set to zero, justify max/min.

Probability Functions

A discrete random variable lists separate possible values, and a probability function assigns probabilities to those values.

$$0 \leq P(X=x) \leq 1 \qquad \sum P(X=x)=1$$

Exam habit: Always check both conditions: probabilities must be possible values, and the whole distribution must add to 1.

RV Easy
RV Medium
RV Hard
Expected Value

The expected value is the long-run average value of a discrete random variable. It is a weighted mean using probabilities as weights.

$$E(X)=\sum xP(X=x)$$

Exam habit: In worded money/game questions, decide whether the variable is the prize, the cost, or the net gain before calculating.

RV Easy
RV Medium
RV Hard
Variance and Standard Deviation

Variance and standard deviation measure how spread out the values are around the mean.

$$\operatorname{Var}(X)=E(X^2)-[E(X)]^2 \qquad \sigma=\sqrt{\operatorname{Var}(X)}$$

Exam habit: Variance is in squared units; standard deviation is in the original units.

RV Easy
RV Medium
RV Hard
Cumulative Probabilities and Transformations

Cumulative probabilities add up values up to a point. Linear transformations change the centre and spread in predictable ways.

$$F(x)=P(X\leq x) \qquad E(aX+b)=aE(X)+b \qquad Var(aX+b)=a^2Var(X)$$

Exam habit: Adding shifts the mean only. Multiplying changes both the mean and the spread.

RV Easy
RV Medium
RV Hard
Bernoulli and Binomial Distributions

A Bernoulli variable records one success/failure trial. A binomial variable counts successes across a fixed number of independent Bernoulli trials.

$$X\sim B(n,p),\quad P(X=x)=\binom{n}{x} p^x(1-p)^{n-x},\quad E(X)=np,\quad Var(X)=np(1-p)$$

Exam habit: Before using binomial, check fixed number of trials, independent trials, two outcomes, and constant success probability.

Log Meaning and Log Laws

A logarithm answers an exponent question. Log laws let you rewrite products, quotients and powers so expressions become easier to solve or simplify.

$$a^c=b \Longleftrightarrow \log_a b=c$$
$$\log_a(MN)=\log_aM+\log_aN \quad \log_a\left(\frac MN\right)=\log_aM-\log_aN \quad \log_a(M^k)=k\log_aM$$

Exam habit: Log laws only combine logs with the same base, and every log argument must be positive.

log Easy
log Medium
log Hard
Solving Logarithmic and Exponential Equations

Log equations usually need one of two moves: convert between log and index form, or combine logs first and then convert.

$$\log_a x=c \Longleftrightarrow x=a^c \qquad a^x=b \Longleftrightarrow x=\frac{\log b}{\log a}$$

Common trap: Solving may create invalid answers. Always check that every log argument is positive in the original equation.

log Easy
log Medium
log Hard
Graphs and Transformations

The graph of a logarithm is the inverse shape of an exponential. Transformations change its domain, vertical asymptote and intercepts.

$$y=\log_a(x-h)+k \quad \Rightarrow \quad \text{vertical asymptote }x=h,\; \text{domain }x>h$$

Exam habit: Find the vertical asymptote by setting the log argument equal to zero.

log Easy
log Medium
log Hard
Logarithmic Modelling and Scales

Logarithms appear in models where multiplicative changes become additive, such as pH, decibels, magnitude scales and solving exponential growth or decay.

$$y=Ae^{kt} \Rightarrow \ln\left(\frac{y}{A}\right)=kt$$

Exam habit: Isolate the exponential or ratio before taking logarithms.

log Easy
log Medium
log Hard
Natural Log Calculus

Natural logarithms connect directly to calculus: the derivative of $\ln x$ is $1/x$, and the integral of $1/x$ is $\ln|x|+c$.

$$\frac{d}{dx}\ln x=\frac1x \qquad \frac{d}{dx}\ln(f(x))=\frac{f'(x)}{f(x)} \qquad \int \frac{f'(x)}{f(x)}dx=\ln|f(x)|+c$$

Exam habit: Look for a numerator that is the derivative of the denominator.

Probability Density Functions

For a continuous random variable, probability is area under the density curve. A single exact value has probability zero.

$$P(a\lt X\lt b)=\int_a^b f(x)\,dx \qquad \int_{-\infty}^{\infty}f(x)\,dx=1 \qquad f(x)\geq0$$

Exam habit: Check the support interval first. Integrate only where the PDF is defined.

CRV Easy
CRV Medium
CRV Hard
Uniform Continuous Distributions

A uniform continuous random variable is equally likely across an interval, so probabilities are proportional to interval lengths.

$$X\sim U(a,b) \Rightarrow f(x)=\frac1{b-a},\;a\leq x\leq b$$
$$E(X)=\frac{a+b}{2} \qquad Var(X)=\frac{(b-a)^2}{12}$$
CRV Easy
CRV Medium
CRV Hard
CDFs and Quantiles

A CDF accumulates area from the left. Differentiate a CDF to recover the PDF, and solve CDF equations to find medians or quantiles.

$$F(x)=P(X\leq x)=\int_{-\infty}^{x}f(t)\,dt \qquad f(x)=F'(x)$$

Exam habit: For continuous variables, $P(X\leq x)=P(X\lt x)$ because $P(X=x)=0$.

CRV Easy
CRV Medium
CRV Hard
Expected Value, Variance and Standard Deviation

The mean is the balance point of the density. Variance and standard deviation measure spread.

$$E(X)=\int_{-\infty}^{\infty}x f(x)\,dx \qquad Var(X)=E(X^2)-[E(X)]^2$$
$$E(X^2)=\int_{-\infty}^{\infty}x^2 f(x)\,dx \qquad \sigma=\sqrt{Var(X)}$$
CRV Easy
CRV Medium
CRV Hard
Linear Transformations and Model Judgement

Changing units or adding fixed costs transforms a random variable. You should also be able to judge whether a continuous model matches a context.

$$Y=aX+b \Rightarrow E(Y)=aE(X)+b,\; Var(Y)=a^2Var(X),\; SD(Y)=|a|SD(X)$$

Exam habit: Adding affects the mean, not the spread. Multiplying scales the spread.

Sample Proportion and Sampling Distribution

Before building a confidence interval, students need to understand $\hat p$ as a random variable from a random sample.

$$\hat p=\frac{x}{n} \qquad E(\hat p)=p \qquad SD(\hat p)=\sqrt{\frac{p(1-p)}{n}}$$

Exam habit: Use proportions as decimals when calculating, then convert to percentages only in the final interpretation if needed.

CI Easy
CI Medium
CI Hard
Constructing Confidence Intervals

A confidence interval estimates the unknown population proportion using a sample proportion and a margin of error.

$$\hat p\pm z^*\sqrt{\frac{\hat p(1-\hat p)}{n}}$$

Exam habit: Use proportions as decimals when calculating, then convert to percentages only in the final interpretation if needed.

CI Easy
CI Medium
CI Hard
Interpreting Intervals and Claims

Confidence intervals are used to describe plausible values for $p$ and to judge whether a claim is supported by sample evidence.

$$\text{CI}=(\hat p-E,\hat p+E) \qquad \text{claim value inside? outside?}$$

Exam habit: Use proportions as decimals when calculating, then convert to percentages only in the final interpretation if needed.

CI Easy
CI Medium
CI Hard
Margin of Error and Sample Size

The width of a confidence interval depends on the critical value, the sample proportion and the sample size.

$$E=z^*\sqrt{\frac{\hat p(1-\hat p)}{n}} \qquad n\geq\frac{(z^*)^2\hat p(1-\hat p)}{E^2}$$

Exam habit: Use proportions as decimals when calculating, then convert to percentages only in the final interpretation if needed.

CI Easy
CI Medium
CI Hard
Reverse Confidence Interval Problems

Harder interval questions ask you to recover hidden information, compare methods, or explain whether the sampling process is trustworthy.

$$\hat p=\frac{L+U}{2} \qquad E=\frac{U-L}{2} \qquad z^*=\frac{E}{SE}$$

Exam habit: Use proportions as decimals when calculating, then convert to percentages only in the final interpretation if needed.

Question 12 — A machine dispenses udon noodles where $X \sim N(\mu, \sigma^2)$, with $P(X \leq 150) = 0.0228$ and $P(X \geq 165) = 0.1587$.
(a)
Determine the mean and standard deviation of the mass dispensed.
3 marks
(b)
A tray contains 20 packets. A serving is underweight if $X < 150$ g. Find the probability that a tray contains no underweight servings.
3 marks
(c)
A sample of 200 servings gives the confidence interval $(0.0651, 0.1349)$ for the proportion $p$ of underweight servings. Determine the margin of error.
1 mark
(d)
Determine the confidence level used to calculate this interval.
3 marks
3:00
Answer unlocks after 3 minutes
⏳ Think it through first — no peeking!
STEP-BY-STEP PROCESS
→(a): $P(X \leq 150) = 0.0228$. Recognise $0.0228 \approx 2.2\%$ which corresponds to $z = -2$ in the standard normal. So $\dfrac{150 - \mu}{\sigma} = -2$.
→$P(X \geq 165) = 0.1587 \approx 15.87\%$, so $P(X \leq 165) = 0.8413$, which gives $z = +1$. So $\dfrac{165-\mu}{\sigma} = 1$.
→You now have two equations in $\mu$ and $\sigma$. Solve simultaneously.
→(b): Find $P(X < 150)$ from your $\mu, \sigma$. Each packet is independent. 20 packets — this is a binomial probability $P(\text{all good}) = P(X \geq 150)^{20}$.
→(c): Margin of error = half the width of the CI.
→(d): $\hat{p}$ is the midpoint. Use $E = z^* \sqrt{\hat{p}(1-\hat{p})/n}$ to find $z^*$, then look up the confidence level.
HINT
Click hint to reveal
FULL SOLUTION
aFrom $z$-scores: $150 = \mu - 2\sigma$ and $165 = \mu + \sigma$.
Subtracting: $15 = 3\sigma \Rightarrow \sigma = 5$. Then $\mu = 165 - 5 = 160$.
$\mu = 160$ g, $\sigma = 5$ g
b$P(X < 150) = P(Z < -2) = 0.0228$, so $P(X \geq 150) = 0.9772$.
$P(\text{no underweight in 20}) = 0.9772^{20} \approx 0.632$
cMargin of error $= \dfrac{0.1349 - 0.0651}{2} = \dfrac{0.07}{2} = 0.035$
d$\hat{p} = \dfrac{0.0651 + 0.1349}{2} = 0.1$. $n = 200$.
$E = 0.035 = z^*\sqrt{0.1 \times 0.9 / 200} = z^* \times 0.02121$
$z^* = 0.035/0.02121 \approx 1.65$, which corresponds to a 90% confidence level.
